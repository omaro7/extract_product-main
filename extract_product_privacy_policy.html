제품 추출기 개인정보처리방침

최종 업데이트: 2025년 4월 1일

물개발자/Hoodpub("당사")는 사용자의 개인정보 보호를 중요하게 생각합니다. 이 개인정보처리방침은 Chrome 확장 프로그램 '제품 추출기'("확장 프로그램")가 어떤 정보를 수집하고, 이를 어떻게 사용하며, 보호하는지에 대한 정보를 제공합니다.

1. 수집하는 정보

당사의 확장 프로그램은 다음과 같은 제품 정보만을 수집합니다:

  • 제품 제목
  • 제품 이미지 URL
  • 제품 페이지 URL
  • 제품 ID
  • 제품 가격

당사는 사용자의 검색 기록, 브라우징 패턴 또는 기타 개인 식별 정보를 수집하지 않습니다.

2. 정보 수집 방법

확장 프로그램은 사용자가 명시적으로 버튼을 클릭하거나 단축키를 사용하여 제품 정보 추출을 요청할 때만 정보를 수집합니다. 자동으로 정보를 수집하지 않습니다.

3. 정보 사용 방법

수집된 제품 정보는 사용자가 제품을 비교하고 참조할 수 있도록 하는 목적으로만 사용됩니다. 이 정보는 사용자의 브라우저 로컬 스토리지에만 저장되며, 당사나 제3자의 서버로 전송되지 않습니다.

4. 정보 저장 및 보안

모든 수집된 데이터는 사용자의 브라우저 내에만 로컬로 저장됩니다. 클라우드나 외부 서버에 백업되지 않으며, 인터넷 연결 없이도 저장된 정보에 접근할 수 있습니다.

데이터는 사용자가 확장 프로그램을 제거하거나 브라우저 데이터를 삭제할 때까지 유지됩니다.

5. 지원하는 쇼핑몰

현재 이 확장 프로그램은 쿠팡 쇼핑몰만 지원합니다. 추후 사용자 요청에 따라 다른 쇼핑몰로 지원을 확장할 계획이 있습니다.

6. 권한 요청

이 확장 프로그램은 다음과 같은 권한을 요청합니다:

  • 특정 웹사이트에 대한 접근 권한 (쿠팡): 제품 정보를 추출하기 위함
  • 스토리지 접근 권한: 추출한 제품 정보를 로컬에 저장하기 위함

확장 프로그램은 필요한 최소한의 권한만을 요청합니다.

7. 제3자 서비스

이 확장 프로그램은 어떠한 제3자 분석 도구나 서비스도 사용하지 않습니다.

8. 개인정보처리방침 변경

당사는 이 개인정보처리방침을 필요에 따라 업데이트할 수 있습니다. 중요한 변경사항이 있을 경우, 확장 프로그램을 통해 새로운 버전의 개인정보처리방침에 대해 알려드립니다.
