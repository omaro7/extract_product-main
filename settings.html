쿠팡 제품 크롤러 설정

keyboard 키보드 단축키

이 확장 프로그램은 다음과 같은 키보드 단축키를 제공합니다:

Alt+C - 제품 정보 수집하기

쿠팡 웹사이트에서 이 단축키를 사용하면 현재 페이지의 제품 정보를 자동으로 수집합니다.

Chrome은 사용자가 모든 확장 프로그램의 단축키를 직접 설정할 수 있도록 합니다. 단축키를 변경하려면:

  1. Chrome 브라우저에서 chrome://extensions/shortcuts로 이동하세요.
  2. '쿠팡 제품 크롤러' 확장 프로그램을 찾으세요.
  3. 원하는 단축키로 변경하세요.
open_in_new 단축키 설정 페이지로 이동
